Asserts that two expressions are equal to each other.

On panic, this macro will print the values of the expressions with their debug representations.

Examples

fn main() { let a = 3; let b = 1 + 2; assert_eq!(a, b); }
let a = 3;
let b = 1 + 2;
assert_eq!(a, b);
