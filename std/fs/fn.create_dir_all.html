pub fn create_dir_all<P: AsRef<Path>>(path: P) -> Result<()>

Recursively create a directory and all of its parent components if they are missing.

Errors

This function will fail if any directory in the path specified by path does not already exist and it could not be created otherwise. The specific error conditions for when a directory is being created (after it is determined to not exist) are outlined by fs::create_dir.
