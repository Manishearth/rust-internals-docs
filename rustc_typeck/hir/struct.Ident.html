impl Ident

fn from_name(name: Name) -> Ident

Unstable (rustc_private)

Creates a HIR identifier with both name and unhygienic_name initialized with the argument. Hygiene properties of the created identifier depend entirely on this argument. If the argument is a plain interned string intern("iter"), then the result is unhygienic and can interfere with other entities named "iter". If the argument is a "fresh" name created with gensym("iter"), then the result is hygienic and can't interfere with other entities having the same string as a name.
