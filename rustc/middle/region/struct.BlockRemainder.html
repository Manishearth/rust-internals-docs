Unstable (rustc_private)

Represents a subscope of block for a binding that is introduced by block.stmts[first_statement_index]. Such subscopes represent a suffix of the block. Note that each subscope does not include the initializer expression, if any, for the statement indexed by first_statement_index.

For example, given { let (a, b) = EXPR_1; let c = EXPR_2; ... }:

  • the subscope with first_statement_index == 0 is scope of both a and b; it does not include EXPR_1, but does include everything after that first let. (If you want a scope that includes EXPR_1 as well, then do not use CodeExtentData::Remainder, but instead another CodeExtent that encompasses the whole block, e.g. CodeExtentData::Misc.
