Creates a cause for obligations that are derived from obligation by a recursive search (e.g., for a builtin bound, or eventually a impl Foo for ..). If obligation is itself a derived obligation, this is just a clone, but otherwise we create a "derived obligation" cause so as to keep track of the original root obligation for error reporting.
